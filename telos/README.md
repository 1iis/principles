# Telos
> What we aim for

The direction: what our methods are meant to make more likely over time.

We aim for [augmented theory-building](augmented-theory-building.md): using machines to improve understanding, not merely output. That should produce [intelligible systems](intelligible-systems.md), build [compound capability](compound-capability.md), and happen inside [literate systems](literate-systems.md) where intent, code, data, outputs, and context belong together in the medium. These threads converge in [the new personal computer](the-new-personal-computer.md): context and capabilities organized around human intent.
