# Ethos
> What we value

Our stance: the values, tastes, standards, and instincts we try to preserve before any particular method or goal.

We start with [human agency](human-agency.md): the tool is a lever, and responsibility remains with the operator. From there, [understanding is the work](understanding-is-the-work.md): research, judgment, and theory-building come before code when the problem is unclear. The artifact we want is [boring](boring-is-good.md) in the best sense: intelligible, maintainable, and readable without an oracle.
