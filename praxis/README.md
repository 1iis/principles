# Praxis
> How we work

The playbook by which principles become work.

We [research as signal](research-as-signal.md), then go [slow before fast](slow-then-fast.md): first acquire enough understanding, then execute with narrower ambiguity. We keep [exploration and artifact](exploration-and-artifact.md) distinct, practice [context discipline](context-discipline.md), use the [generated buffet](generated-buffet.md) for selection rather than accumulation, build through [composable boundaries](composable-boundaries.md), and write [literate source](literate-source.md) so the why stays near the how.
